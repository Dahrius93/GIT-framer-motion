# 🍕 Ninja Joint - Pizza Ordering App

A modern, animated pizza ordering application built with React, featuring smooth transitions and an intuitive user experience.

![React](https://img.shields.io/badge/React-16.13.1-blue.svg)
![Framer Motion](https://img.shields.io/badge/Framer%20Motion-1.10.3-purple.svg)
![React Router](https://img.shields.io/badge/React%20Router-5.1.2-red.svg)

[deploy progetto su questo link](https://pizza-order-motion.netlify.app/)

## ✨ Features

- **Interactive Pizza Builder**: Customize your pizza with different bases and toppings
- **Smooth Animations**: Powered by Framer Motion for delightful user interactions
- **Responsive Design**: Works seamlessly across desktop and mobile devices
- **Multi-step Ordering Process**: Guided flow from base selection to order confirmation
- **Modal Confirmations**: Elegant order confirmation dialogs

## 🚀 Technologies Used

- **React 16.13.1** - UI library for building user interfaces
- **Framer Motion 1.10.3** - Animation library for React
- **React Router DOM 5.1.2** - Declarative routing for React
- **Create React App** - Build setup and development server

## 📋 Prerequisites

Before running this project, make sure you have the following installed:

- **Node.js** (version 14 or higher)
- **npm** or **yarn** package manager

## 🛠️ Installation

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/ninja-joint.git
   cd ninja-joint
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**

   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open your browser**

   The app will be available at [http://localhost:3000](http://localhost:3000)

## 📖 Usage

1. **Home Page**: Start your pizza ordering journey
2. **Base Selection**: Choose your preferred pizza base
3. **Toppings**: Select your favorite toppings
4. **Order Review**: Review your pizza and confirm the order
5. **Confirmation**: Receive order confirmation with smooth animations
